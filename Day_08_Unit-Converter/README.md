# Day 08 — Unit Converter

A command-line unit conversion tool that supports weight and length conversions through a simple menu interface.

---

## Features

- Converts between kilograms and pounds
- Converts between centimetres and inches
- Lists all available conversions on launch
- Accepts a conversion type and value, then prints the result with unit labels

---

## Concepts used

- Functions and dictionary-based operation mapping
- Loops and conditional statements
- User input handling
- Arithmetic and formatted output

---

## How to run

```bash
python main.py
```

### Example

```
Available conversions:
1. kg to pounds
2. pounds to kg
3. cm to inches
4. inches to cm

Select conversion: 1
Enter value: 70
70 kg = 154.32 pounds
```
