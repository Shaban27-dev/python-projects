⏳🍅 Day 24 — Pomodoro Timer (Tkinter GUI)

A GUI-based Pomodoro Timer built using Tkinter that helps manage work and break sessions efficiently using the Pomodoro technique 🎯.

🧾 Features

▶️ Start timer for work and break sessions

🔁 Automatic cycle: Work → Short Break → Long Break

⏱️ Countdown timer with minute & second display

🎨 Dynamic UI updates (Work/Break colors)

✅ Checkmarks to track completed work sessions

🔄 Reset button to restart the timer

🧠 Concepts Used

Concepts used: 🖼️ tkinter GUI, ⏱️ time-based events (after), 🔁 loops, 🧩 functions, 🧮 math module, 🎨 UI updates, ⌨️ event-driven programming.
