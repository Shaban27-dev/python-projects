# Day 09 — Bank Account Simulator (OOP)

An object-oriented bank account system built with Python classes. Supports account creation, deposits, withdrawals, and a full transaction log.

---

## Features

- Create a bank account with an opening balance
- Deposit money into the account
- Withdraw money with validation against insufficient funds
- View the complete transaction history
- Displays updated balance after every operation

---

## Concepts used

- Object-oriented programming — classes and objects
- Methods and encapsulation
- Conditional statements for validation
- User input handling

---

## How to run

```bash
python main.py
```

### Example

```
Account created for: John | Balance: $1000.00

1. Deposit
2. Withdraw
3. Transaction History
4. Exit

Select option: 1
Enter deposit amount: 500
Deposited $500.00. New balance: $1500.00
```
