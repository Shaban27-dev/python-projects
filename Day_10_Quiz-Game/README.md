# Day 10 — Quiz Game

A command-line trivia quiz that asks five general knowledge questions, checks each answer, gives instant feedback, and shows a final score with percentage.

---

## Features

- 5 general-knowledge questions per session
- Instant correct / wrong feedback after each answer
- Reveals the correct answer on a wrong guess
- Calculates and displays final score and percentage

---

## Concepts used

- Dictionaries for question and answer storage
- Loops for question iteration
- String comparison for answer checking
- Score calculation and formatted output

---

## How to run

```bash
python main.py
```

### Example

```
Q1: What is the capital of France?
Your answer: Paris
Correct!

Q2: How many continents are there?
Your answer: 6
Wrong! The correct answer is: 7

...

Your score: 4/5 (80%)
```
