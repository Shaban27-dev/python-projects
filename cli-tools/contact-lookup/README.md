# Day 07 — Contact Information Lookup

A command-line contact directory that stores names, phone numbers, and email addresses. Users can search for a contact by name and retrieve their details instantly.

---

## Features

- Stores multiple contacts with name, phone, and email
- Search by name — shows full contact details if found
- Displays a clear error message if the contact does not exist
- Menu-driven interface for add, search, and delete operations

---

## Concepts used

- Dictionaries for structured data storage
- Functions for modular design
- User input handling
- Conditional statements and data retrieval

---

## How to run

```bash
python main.py
```

### Example

```
1. Add Contact
2. Search Contact
3. Delete Contact
4. Exit

Select option: 2
Enter name to search: Alice
Name:  Alice
Phone: 9876543210
Email: alice@example.com
```
