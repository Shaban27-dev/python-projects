# Day 01 — Simple Calculator

A basic command-line calculator that takes two numbers and an operator, performs the operation, and prints the result.

---

## Features

- Supports addition, subtraction, multiplication, and division
- Handles division by zero with a clear error message
- Validates non-numeric input before processing
- Loops until the user chooses to exit

---

## Concepts used

- Variables and data types
- `input()` / `print()` for I/O
- Conditional statements (`if / elif / else`)
- Basic error handling

---

## How to run

```bash
python main.py
```

### Example

```
Enter first number: 10
Enter operator (+, -, *, /): /
Enter second number: 2
Result: 10.0 / 2.0 = 5.0
```
