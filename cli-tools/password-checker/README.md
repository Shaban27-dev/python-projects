# Day 02 — Password Strength Checker

A command-line tool that evaluates the strength of a password based on length and character composition rules.

---

## Features

- Checks password length (minimum threshold)
- Detects presence of uppercase letters, lowercase letters, digits, and special characters
- Returns a strength rating: **Weak**, **Medium**, or **Strong**
- Provides feedback on what the password is missing

---

## Concepts used

- Strings and string methods
- Conditional statements
- Input validation
- Basic loops

---

## How to run

```bash
python main.py
```

### Example

```
Enter your password: Hello123
Strength: Medium
Tips: Add special characters to make it stronger.
```
